# Andreani — Vistas para Clientes

Proyecto separado del catálogo interno. Solo contiene vistas protegidas
por contraseña para compartir con clientes externos.

## Rutas
- `/modelo-1` → Vista del Modelo 1 (Distribución general)

## Agregar un nuevo modelo
1. Copiar `src/pages/Modelo1View.jsx` como base
2. Ajustar el ID de modelo, el flujo de etapas y el código de acceso si corresponde
3. Agregar la ruta en `src/App.jsx`
